How Permissions Work

Roles are assigned by an admin from the browser DevTools console while on any page of this site:

assignRole('discord:YOUR_USER_ID', 'staff')

Replace YOUR_USER_ID with the user's numeric ID shown after they sign in (visible in the console as part of their UID). Valid roles: user · staff · admin
