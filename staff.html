How Permissions Work

Roles are stored in Supabase table public.user_roles. Admins can still assign roles from DevTools using:

assignRole(getSession().uid, 'staff')

To assign someone else, use their full session UID (example: google:uuid... or discord:uuid...). Valid roles: user · staff · admin

First-time setup requires one admin row inserted in Supabase SQL (see supabase-schema.sql bootstrap comment).
